## App Web com Asp.net MVC + API Rest

### Stacks:

* .NET
* C#
* Asp.net
* Entity Framework
* Sql Server


### Run project in Local:

* Clone repository, go to root folder and run:
```
$ dotnet restore
```
* To see App Web run:
```
$ dotnet run
```
